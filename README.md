# learning machine larning with python

notebook of personal learning of machine learning with python

## requirements

* OS - unix base system
* docker >= 19.03
* docker-compose >= 1.26.0

## build and run

```
$ docker-compose build
$ docker-compose run app python3 src/path/to/main.py
```
